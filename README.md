# MobileApps2025-2301321013



## Идея

Приложението е прост нотепад за организиране на бележки. Потребителят може да добавя, търси и сортира бележки по категории, като има основен фокус върху бързина и удобство. Поддържа тъмен и светъл режим за по-добра четливост.



## Как работи

Приложението използва локална Room база данни за съхранение на бележки. Всяка бележка има име, съдържание, категория, дата на създаване и дата на последна редакция. Потребителят може да добавя нови бележки, да търси по съдържание или категория и да преглежда съществуващите. Интерфейсът е интуитивен, с Material 3 компоненти и превключване между светъл и тъмен режим. Всички промени се запазват след рестарт.



## Архитектура

MVVM + Repository слой. ViewModel управлява UI логиката, Repository достъпва Room базата. LiveData наблюдава промени в данните.



## Потребителски поток

1. Стартиране на приложението  

2. Преглед на списък с бележки  

3. Добавяне или редактиране на бележка  

4. Търсене или филтриране по категория  

5. Бележките се запазват автоматично  



## Стъпки за стартиране

1. Клонирайте репото  

2. Отворете проекта в Android Studio  

3. Build \& Run на емулатор или устройство
